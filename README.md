# Student-Toolkit
A student tool kit made entirely using python and some external libraries such as custom tkinter and QR generator.
It is a Student Toolkit made to help students as a one stop shop for every academic need on can have. 
External libraries are : CustomTkinter QRcode generator.
One can enter the their ISA and ESA marks and create get teh grade they might get by the end of the semester.
They can also record the amount of time they are dedicating towards a every subject for a healthy balance.
You can also get every resource one would need for the preperation for end and mid semister exams.
You can also calculate your attendance and how many classes u need to attened in a week to attain said attendance percentage.
You can also keep record of all the tasks you need to complete as a todo-list.
